// Rankine · Coulomb — Presiones Activas, Pasivas y en Reposo

Teoria de Rankine — Supone muro liso (sin friccion muro-suelo) y relleno horizontal. Calcula presiones activas (Ka) y pasivas (Kp) para suelos con cohesion y friccion.
Ka = tan²(45 - phi/2) · Kp = tan²(45 + phi/2)
sigma_a = Ka·gamma·z - 2c·sqrt(Ka) · sigma_p = Kp·gamma·z + 2c·sqrt(Kp)

// Diagrama de Presiones
Teoria de Coulomb — Considera friccion entre el muro y el suelo (delta) y relleno inclinado (beta). Mas precisa que Rankine para muros rugosos. Solo calcula presion activa.
Ka = sin²(phi+alpha) / [sin²(alpha)·sin(alpha-delta)·(1+sqrt(sin(phi+delta)·sin(phi-beta)/sin(alpha-delta)·sin(alpha+beta)))²]
